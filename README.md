# 1 任务代码流程

## 1.1 故障保护任务（FaultProtect）

**功能：**故障保护

**优先级：**6，**周期**：200ms

3种保护状态：监控保护状态、等待解除保护状态、解除保护状态；创建软件定时器，用于计算解除保护等待时间

使用外部中断监听ALERT引脚，一旦引脚拉高，获取BQ76940系统寄存器（SYS_STAT）状态

**主循环：**

**（1）**   **监控保护状态**

a)    **软件保护**

充电模式监控保护：过流、过温、低温

一旦超过阈值关闭充电MOS管，置对应故障标志位

放电模式监控保护：过温、低温

一旦超过阈值关闭放电MOS管，置对应故障标志位

b)    **硬件保护（BQ76940****芯片）（外部中断引脚（ALERT****）触发）**

放电过流保护：检查放电过流标志位（FlAG_ALERT_OCD）

放电短路保护：检查放电短路标志位（FlAG_ALERT_SCD）

充电过压保护：检查充电过压标志位（FlAG_ALERT_OV）

放电欠压保护：检查放电欠压标志位（FlAG_ALERT_UV）

**（2）**   **等待解除保护状态**

短路（60s）：

充电：过流（60s）、欠压、过温、低温

重新开启放电MOS管

充电：过流（60s）、过压、过温、低温

重新开启充电MOS管

**（3）**   **解除保护状态**

根据用户设置重新开启充放电开关

## 1.2 数据采样任务（DataAcqTask）

**功能**：电芯电压与电流采样

**优先级：**5，**周期**：250ms

**主循环：**

（1）   采集单体电芯电压（250ms周期）

（2）   采集电池组总电压（250ms周期）

（3）   采集电池温度（2s周期）

（4）   采集电池电流（250ms周期，软件触发）（可选硬件中断触发：每次下载都需给BQ76940芯片重新上下电）

（5）   计算电压差、平均电压、实时功率、最大和最小电压

## 1.3 状态控制任务（StateControl）

**功能：**充放电状态识别与逻辑控制

**优先级**：4，**周期**：250ms

**主循环：**

（1）   待机状态：-50mA <电流< 50mA

检查用户是否启用了充电开关：根据SOC值是否开启充电：开启充电MOS开关

用户是否启用了放电开关：根据SOC值是否开启放电：开启放电MOS开关

（2）   充电状态：

条件：电流> 50mA

SOC >= 停止充电SOC阈值：关闭充电MOS开关

（3）   放电状态：

条件：电流< -50

SOC <= 停止放电SOC阈值：关闭放电MOS开关

（4）   故障状态：检查到故障码

（5）   睡眠状态：

## 1.4 电量管理任务（BatteryGauge）

**功能**：SOC计算和电量管理

**优先级**：3，**周期**：1000ms

**主循环：**

（1）    容量和SOC上电初始化：计算SOC；电池包实时容量=电池包额定容量；剩余容量=实际容量*SOC

（2）    温度补偿算法：根据温度变化情况校准实时容量和剩余容量

（3）    SOC计算（开路电压法 + 安时积分法）：

l睡眠模式使用“开路电压法”计算SOC和剩余容量：通过对最低电芯电压进行三元锂电池OCV查表（101个数据点）得到SOC值；剩余容量=实际容量*SOC

待机模式、充电模式、放电模式使用“安时积分法”计算剩余容量和SOC

## 1.5 通信任务（CommTask）

**功能**：CAN总线数据收发，支持外部设备交互

**优先级**：2，**周期**：队列接收阻塞

**主循环**：

osMessageQueueGet(canRxMQHandle, &can_rx_msg, NULL, 2000);

（1）    接收到数据

接收逻辑：解析指令

（2）    未接收到数据

发送逻辑：发送电池信息

## 1.6 均衡控制任务（BalanceControl）

**功能：**能量管理

**优先级**：1，**周期**：250ms

**主循环：**

（1）     开始均衡条件：按顺序判断，**同时满足**

用户级条件：开启均衡功能

系统级条件：

1st. 充电状态

2nd. 非均衡状态

3rd. 最小电芯电压>=3.40V、最大压差>=50mV

4th. 连续4次满足此条件

**开始均衡**：查找符合条件的电芯，开启均衡，记录均衡电芯信息

（2）停止均衡条件：首先判断是否处于均衡状态，满足以下其中一点，停止均衡

用户级条件：禁用均衡功能

系统级条件：

​	非充电状态

​	均衡状态

​	最小电芯电压<3.40V、最大压差<30mV

​	连续4次满足此条件

电池均衡管理（避免相邻电芯同时均衡）
